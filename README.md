# My First Hero App

## Description
My First Hero is a fun and interactive web app designed for users who love creativity and adventure. In this app, you can create and customize your very own hero characters, choosing from a variety of costumes, accessories, and special abilities. Once your hero is ready, embark on exciting missions and challenges that test your strategy and decision-making skills.

The app features a colorful, engaging interface that is easy to navigate for users of all ages, with animations and visual effects that bring your hero’s world to life. As you progress, you can unlock achievements, earn rewards, and discover new customization options to make your hero truly unique.

Whether you’re exploring, experimenting, or competing in missions, My First Hero encourages creativity, problem-solving, and endless fun. It’s the perfect app for anyone who wants to step into the shoes of a hero and create their own adventure!

## Technologies Used
- **React.js** – Component-based UI development
- **React Router** – Navigation and routing between pages
- **Tailwind CSS** – Utility-first styling for fast and responsive design
- **DaisyUI** – Tailwind CSS component library for UI elements
- **HTML5** – Structure of the app
- **Axios** – HTTP requests and API handling
- **React Toaster** – Notifications and alerts
